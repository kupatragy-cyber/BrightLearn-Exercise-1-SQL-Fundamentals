# BrightLearn-Exercise-1-SQL-Fundamentals
BrightLearn Exercise 1 SQL Fundamentals
SQL EXERCISE README

This assignment contains basic SQL queries used to retrieve and filter data from a table called "employees".

OBJECTIVES:

* Learn how to use SELECT to retrieve data
* Use WHERE to filter records
* Use DISTINCT to remove duplicates
* Sort data using ORDER BY
* Limit results using LIMIT
* Combine conditions using AND, OR, NOT, and IN

TABLE USED:
employees

COLUMNS:
id, first_name, last_name, department, salary, hire_date, city

DESCRIPTION:
The employees table stores information about employees including their names, departments, salaries, and locations.

KEY CONCEPTS:

* SELECT: used to choose columns
* FROM: specifies the table
* WHERE: filters rows
* ORDER BY: sorts results
* LIMIT: restricts number of rows

SQL ORDER:
SELECT → FROM → WHERE → ORDER BY → LIMIT

NOTES:

* Text values must be in quotes ('IT')
* Use = for equal
* Use <> for not equal
* Use IN for multiple values
* Use AND when all conditions must be true
* Use OR when any condition can be true

CONCLUSION:
This exercise helped build a basic understanding of SQL queries and how to work with data in a table.
